# B2B SaaS YouTube Content Strategy Research

## Project Overview
Repository ini berisi kumpulan data riset intensif mengenai taktik dan arsitektur konten YouTube khusus untuk industri B2B SaaS.

## Kenapa Memilih Pakar Ini?
Para pakar yang dipilih (seperti Tyler Lessard dan TK Kader) adalah praktisi asli yang bekerja langsung di industri software, bukan sekadar teoritikus.

## Folder Structure
- `/research/sources.md`: Daftar komprehensif 10 expert dan anotasi.
- `/research/linkedin-posts/`: Kumpulan insights taktis dari platform LinkedIn.
- `/research/youtube-transcripts/`: Transkrip penuh video YouTube yang diekstrak melalui API.
